Cuisine
Features
Initially on click on the cuisine tab I am displaying italian cuisine's images returned from the API. I am displaying them in an Image Accordian.
Then on click on the mexican, indian or american cuisines I am displaying its recipes images in an Image Accordian. I have displayed recipe title using the caption of the figure.

On click on the 'View Recipe' button I am displaying recipe details of the image currently displaying in image accordian. In recipe details I am additionally showing the rating of the recipe.
In a tab menu of recipe details a 'Similar Video' tab will appear if there would be a video from YouTube API request, otherwise won't.
Learning
I have learnt to make an Image accordian. It is useful to display images in a limited space. And showing the glimpes of the next image so user could jump on to others if want to.
I have learnt to use figcaption tag to show caption of the images and also learnt css transitions.

For the rating I am getting response value as a number (ie. 3,4). I have converted this text value in to an image and displayed it.

I have learnt to make YouTube API request. And to integrate returned video in an iframe.
I have set iframe's max-width to 100% and on window resizing I am setting its height to 60% of the width to make it responsive.
As the 'Similar Video' tab should not appear when YouTube API doesn't return a video I have learnt to create dynamic css based on returned API response.
